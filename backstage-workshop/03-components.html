Open the Terminal tab from the Pod Detail page

Complete these four commands to register the backend with the gateway component:

  1. Check the backend dataservice for records. The result should be an empty set:

    curl localhost:8080/ws/data/all

  2. Populate the backend with the National Parks dataset:

    curl localhost:8080/ws/data/load

  3. List registered gateway backends. The only result should be the gateway’s built-in "Red Hat Summit POI" dataset:

    curl poi-gateway-app:8080/backend/list

  4. Register the new backend component with the gateway component:

    curl poi-gateway-app:8080/backend/register/poi-backend?endpoint=http://poi-backend-app:8080

Check the map for results!
