Templates

Introducing Templates

Launch the Frontend template

Invoke the Angular Single Page Application frontend template in backstage

Insert your NAMESPACE value (from the previous chapter…​ it should be your username) into the form inputs in place of the <NAMESPACE> string. Click NEXT STEP to proceed.

Insert your NAMESPACE value, then click NEXT STEP to proceed

Review the input selections, click CREATE when ready

Review the template output. Click on the Source Code Repository link to view the newly scaffolded frontend repo

View the Source

Explore the newly created repo, paying special attention to the catalog-info.yaml file

The resulting component yaml is available in the catalog-info.yaml file at the root of the repo.

If you the GitHub org carefully, you will notice a second "gitops" repo has been created. Add "-gitops" to the end of the repo url to explore the contents of the other repo

Launch the Gateway Template

Review the template output. Click on Pipeline status to view the build pipelines for the app.

Tekton Pipelines

Check the status of your pipelines. Select one of your pipeline runs, preferrably one that is active

Select Logs to watch the build process

Monitor the build output. When the build has completed, select Pipelines from the side menu to check the status of the other pipeline run

When both builds have finished successfully, proceed to the Topology view
